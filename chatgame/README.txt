READ ME
Per il corretto funzionamento del programma 
è necessario aver installato python 3.9 ed 
eseguire i file .py in quest'ordine: 
1.generatore_domande.py
2.chatgame_server.py
3.chatgame_client.py
Una volta avviato il server è necessario avviare il/i 
client, inserire l'IP del server che è mostrato a video 
nella GUI del server e premere "AVVIO". 
In seguito si dovrà inserire il nome nell'apposito 
widget e premere il tasto invio. A questo punto il 
client sarà connesso al server, si potrà dunque 
premere start sul server per far partire il gioco.
